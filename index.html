Basics to HTML5

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Here are some simple HTML tags commonly used in websites:

  • <p> - Defines a paragraph.
  • <h1>, <h2>, <h3>, ..., <h6> - Define headings of different sizes.
  • <a> - Defines a hyperlink.
  • <img> - Defines an image.
  • <ul> and <li> - Define an unordered list and list items.
  • <ol> and <li> - Define an ordered list and list items.
  • <div> - Defines a division or a section in an HTML document.
  • <span> - Defines a section in a document.
  • <strong> - Defines text with strong importance.
  • <em> - Defines emphasized text.
